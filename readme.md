# 客户经理与Copilot：科技赋能的一天

**时长**：5-6分钟

**主题**：展现现代银行客户经理如何借助Copilot与AI技术，提升工作效率、应对挑战，为客户提供专业服务。

## 场景描述

场景：会议室、工位、餐厅、Copilot界面。

# 分镜

---

## **8:00 - 8:30：到达办公室，参加晨会** 🌅🏢

### **镜头1：远景，早上8:00**

- **画面**：阳光洒入现代化办公大楼，客户经理推开大门进入办公室。屏幕左下角显示时间标注“8:00”。
- **说明**：远景交代环境，突出科技化的现代办公氛围。

---

### **镜头2：中景，早上8:30** 🖥️👩‍💼👨‍💼

- **画面**：客户经理站在会议室前，通过大屏幕与团队分享当天的工作计划。
- **技术展示**：
    - 大屏幕显示Copilot自动生成的“今日任务概览”：重点客户列表、优先级排序及简要分析。
    - 重点标注：“微软公司 - 资金管理与保险需求分析”。
- **对话**：
    - 客户经理：“今天的主要任务是拜访微软公司。他们计划扩建全球研发中心，并为员工提供更完善的健康保险。Copilot分析表明，这可能是个优质合作机会。”
    - 同事A：“了解了，我们随时准备支持。”

## **8:30 - 10:00：准备客户拜访** 📧📑

    - 数据图表动态显示微软全球化布局对资金灵活性的需求，以及员工健康险需求的行业对比数据。
- **对话**：
    - 客户经理：“微软的全球研发中心扩建需要流动资金支持，同时他们的员工福利提升计划正匹配我们的综合健康险产品。”

---

### **镜头4：近景，早上9:30** 🖱️🔍

- **画面**：客户经理通过Copilot生成本次客户拜访的提案文件。

### **镜头5：客户经理到访微软会议室** 🏢🔗

- **画面**：客户经理步入微软大楼，背景展示微软Logo和现代化的智能办公环境。

        - 电梯内屏幕播放微软AI研发宣传片，增强场景代入感。

- **技术展示**：
    - 客户经理将笔记本连接会议屏幕，展示Copilot生成的“定制化解决方案”：

### **其他建议**

1. **视觉元素搭配**：每个涉及Copilot或AI技术的场景中，增加动态UI设计，增强科技感。
2. **配乐与音效**：技术展示镜头配以轻快现代的科技感音效，与其他部分背景音乐形成对比。
3. **字幕辅助**：关键镜头使用动态字幕、时间线、补充技术功能解说，便于观众理解。
4. **音乐和音效**：选择轻松愉快的背景音乐，增强视频节奏感。关键技术展示时加入科技感音效。
5. **拍摄时间**：每个镜头拍摄时长控制在30秒内，总时长不超过300秒，确保视频紧凑有力。
6. **后勤准备**：准备人员、场地、办公用品、客户资料、笔记本电脑、白板、麦克风、商务装、食品饮品等，确保拍摄顺利。
7. **后期剪辑**：
    - 按照镜头设计顺序剪辑，保持叙事流畅。
    - 添加必要特效和字幕，增强视频效果和观众理解。
    - 使用转场效果连接不同场景，提高视觉连贯性。
